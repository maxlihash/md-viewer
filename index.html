FAQ

    

      Does HarmonyOS support PWA installation?
      
Yes. HarmonyOS 4.0+ browsers support "Add to Home Screen" for PWAs. MD Viewer includes a manifest and service worker for offline use.

    

    

      Are my files uploaded anywhere?
      
No. All processing happens in your browser via the File API and marked.js. Nothing is sent to any server. You can verify by opening DevTools → Network tab — zero requests after page load.

    

    

      What Markdown features are supported?
      
Full GitHub-Flavored Markdown (GFM): headings, lists, tables, task lists, strikethrough, code blocks with language detection, blockquotes, links, images, and inline HTML.

    

    

      Can I use it offline?
      
Yes — after the first visit, the service worker caches the app. Open a local .md file anytime, even without internet.

    

    

      How is this different from opening .md as plain text?
      
Your phone's text viewer shows raw markdown syntax (# Title, **bold**). MD Viewer renders it into properly formatted HTML — headings, bold text, tables, code blocks — exactly how GitHub or Notion would display it.

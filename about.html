About MD Viewer

Built because opening .md files on a phone shouldn't require an app — or an upload.

Why this exists

I use HarmonyOS. When someone sends me a .md file, the phone opens it as raw text — all the # and ** visible, completely unreadable. There's no built-in Markdown viewer. The app store has a few options, but they're heavy, ask for permissions, or upload files to a server.

MD Viewer solves this: drop a file, see it rendered. Nothing leaves your device. Works in any browser — HarmonyOS, Android, iOS, desktop.

How it works

Everything runs client-side:

  • marked.js — GitHub-Flavored Markdown parser (loaded from CDN)
  • File API — reads local files without uploading
  • Service Worker — caches the app for offline use
  • Web App Manifest — enables PWA installation on home screen

Zero server-side processing. Zero tracking. Zero analytics (except what Cloudflare Pages provides by default).

Tech stack

Static HTML + CSS + vanilla JS. Deployed on Cloudflare Pages. No build step, no framework, no database.
